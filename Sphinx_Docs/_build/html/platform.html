Adapter for a DiskCache backed cache

This cache provides a wrapper for a FanoutCache which is backed by SQLite with the same interface as the RedisCacheAdapter. This cache provides automatic sharding and is process and thread-safe; however it should not be used in production for performance reasons.

retry¶
