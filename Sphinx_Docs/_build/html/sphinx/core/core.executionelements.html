create_step(name='', action='', app='', device='', arg_input=None, next_steps=None, risk=0)¶

Creates a new Step object and adds it to the Workflow’s list of Steps.

Args:

name (str, optional): The name of the Step object. Defaults to an empty string. action (str, optional): The name of the action associated with a Step. Defaults to an empty string. app (str, optional): The name of the app associated with the Step. Defaults to an empty string. device (str, optional): The name of the device associated with the app associated with the Step. Defaults

to an empty string.
arg_input (dict, optional): A dictionary of Argument objects that are input to the step execution. Defaults
to None.

next_steps (list[NextStep], optional): A list of NextStep objects for the Step object. Defaults to None. risk (int, optional): The risk associated with the Step. Defaults to 0.

execute(execution_uid, start=None, start_input='')¶

Executes a Workflow by executing all Steps in the Workflow list of Step objects.

Args:
execution_uid (str): The UUID4 hex string uniquely identifying this workflow instance start (str, optional): The name of the first Step. Defaults to None. start_input (str, optional): Input into the first Step. Defaults to an empty string.
