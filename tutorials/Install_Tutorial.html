How to Setup and Install WALKOFF

0) Make sure Python 2.7+, or Python 3.3+ and git are installed.

1) Open a terminal on linux or a command prompt on Windows, and run the command 

 git clone https://github.com/iadgov/WALKOFF.git

2) Change directories to the WALKOFF directory 

 cd WALKOFF

3) Run the command 

 python setup.py install
this command must be run as root (i.e. using sudo, or in an administrative command shell).

4) Run the command 

 pip install -r "requirements.txt"
to install the required libraries -- this command must be run as root (i.e. using sudo, or in an administrative command shell).

6) Run the command 

 python startServer.py

7) Open a browser of your choice, and navigate to the host IP and port specified in the output of the above command to start the server.

The default IP and port is 

127.0.0.1:5000
the default IP and the port can be changed in the data/walkoff.config file.


The default username is 

 admin
the default password is 
 admin
these can be changed upon initial login

Optional - Each app must have a setup.py script inside of its main directory. In order to save the user the effort of manually running the setup.py script for each app, the user can run the command 

 python installDependencies.py -a|--apps AppOne AppTwo ...
this command must be run as root (i.e. using sudo, or in an administrative command shell). Simply provide the installDependencies script with the '-a' or '--apps' flag followed by a list of one or more apps, separated by spaces.
