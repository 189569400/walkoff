System Configuration

  • Open the Settings tab
    All configuration parameters require you to restart the server for changes to take effect.

    Enabling HTTPS

    HTTPS is enabled when certificates are located in the ./certificates. The default certificates names are walkoff.crt and walkoff.key, but they can be changed by modifying the walkoff/config.py file.

    Configuring Redis

    Walkoff's key-value cache defaults to using DiskCache, a SQLite-backed database. While this database is fine for exploring Walkoff, it is better to use Redis. To use Redis,

    1. Install Redis (see official site for documentation)
    2. Install the Redis Python client with pip install redis
    3. Use the settings page to configure Walkoff's Redis connection

    If your Redis database is secured with a password (and it should be), set an environment variable named "WALKOFF_REDIS_PASSWORD" with your redis password before starting Walkoff
