How to Setup and Install WALKOFF

0. Ensure Python 2.7+, or Python 3.3+, pip and git are installed.
1. Open a terminal on linux or a command prompt on Windows, and run the command
 git clone https://github.com/iadgov/WALKOFF.git
2. Change directories to the WALKOFF directory
cd WALKOFF
3. Run the command
pip install -r requirements.txt

to install the required libraries. This command must be run as root (i.e. using sudo, or in an administrative command shell).

4. Each app must have a requirements.txt file inside of its main directory. In order to save the user the effort of manually installing the dependencies for each app, the user can run the command
 python installDependencies.py -a|--apps AppOne AppTwo ...

this command must be run as root (i.e. using sudo, or in an administrative command shell). Simply provide the installDependencies script with the '-a' or '--apps' flag followed by a list of one or more apps, separated by spaces. Alternatively, run the command

python installDependencies.py

This will install the dependencies for every app in the apps folder.

5. Run the command 
 python startServer.py
6. Open a browser of your choice, and navigate to the host IP and port specified in the output of the above command to start the server.

The default IP and port is 

127.0.0.1:5000

the default IP and the port can be changed in the data/walkoff.config file.

The default username is 

 admin

The default password is 

 admin

These can be changed upon initial login.
