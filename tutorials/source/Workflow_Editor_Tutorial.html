Basic Workflow Creation Tutorial

In this tutorial, we will create our first, very basic workflow in order to demonstrate the general features of the workflow editor.

  • Create a new Workflow

    First, begin by clicking the New Workflow button.


  • Enter Playbook and Workflow Name

    If you specify an existing Playbook name, the new Workflow will be saved under it. Enter the Playbook name and workflow name, then click Create New Workflow.


  • Add actions to the workspace

    Let's begin by adding a Hello World action and a Pause action from the HelloWorld app. Expand the HelloWorld app by double-clicking on the name, or clicking on the drop-down arrow. Then, double-click, or click and drag the desired actions into the workspace.
    Ensure that the Hello World action is set as the starting node.


  • Configure actions

    Some actions will have inputs; some required, some optional. In this case, the Pause action requires you to input the time in seconds that it should pause for. Enter 5.


  • Connecting actions together

    Now, let's connect our actions together. When hovering over the top edge of an action, a red dot will appear. Click and drag from the red dot to the top of the next action in the sequence; an arrow should appear, linking the actions together.
    Add another Hello World action after the Pause and connect it as shown.
