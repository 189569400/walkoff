Interface Overview

In this tutorial, we will briefly cover each of the components of the WALKOFF UI.

  • Apps

    The home of custom interfaces for WALKOFF. These can be used to provide visualizations, telemetry, etc.


  • Playbook

    Used to create and edit Workflows, the core of Walkoff's automation and orchestration capabilities.


  • Scheduler

    Create and configure schedules for running workflows.


  • Devices

    Create and configure devices and logins for Apps and Actions to use.


  • Cases

    View, create and configure logging events for your workflows.


  • Settings

    Manage your WALKOFF instance and users.
