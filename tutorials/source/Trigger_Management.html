Triggers Management

  • Add Trigger

    1. Navigate to the trigger page using navigation on the left of the screen
    2. Fill out the trigger form
    3. Click the add Button

  • Edit Trigger

    1. Select the trigger to edit
    2. Click the edit button to prefill the trigger form below
    3. Edit the trigger details
    4. Click "Submit Edits"

  • Delete Trigger

    1. Select the trigger to Delete
    2. Click "Delete Trigger"
