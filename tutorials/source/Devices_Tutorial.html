Devices Tutorial

  • Open the Devices tab
    To add a new device, click on the "+Add Device" button in the top left.
    Select an app and fill out any fields needed. Not all the fields need to be filled out, and if you need to use a field for something other than its intended use, for example putting an API key into the password field, then you can do that too. (Custom device fields coming soon...)
    Click "Add Device" at the bottom on the form to add teh device. The device and all others will be listed in the device management table. you can filter this table by app or search the devices.
    On each device, you can edit or delete a device.
