Devices Tutorial

  • Here we'll add a device for the Walkoff app, which provides interactions with another Walkoff instance. The devices are defined in the app's api.yaml file

    
walkoff: '0.1'
info:
  title: Walkoff
  version: 1.0.0
devices:
  Walkoff:
    description: Walkoff instance
    fields:
      - name: username
        description: Username for Walkoff instance
        placeholder: username
        type: string
        required: true
      - name: password
        description: Password for Walkoff instance
        placeholder: password
        type: string
        required: true
        encrypted: true
      - name: ip
        description: IP Address of the Walkoff instance
        placeholder: ip address
        type: string
        required: true
      - name: port
        description: Port of the Walkoff instance
        placeholder: port
        type: integer
        minimum: 1
        maximum: 65535
        required: true
    Open the Devices tab
    To add a new device, click on the "+Add Device" button in the top left.
    Select an app and fill out any fields needed.
    Click "Add Device" at the bottom on the form to add the device. The device and all others will be listed in the device management table. you can filter this table by app or search the devices.
    On each device, you can edit or delete a device.
