Case Management

  • Open then Cases page. This will display the events logged for a selected case. You can select the case in the search bar above the table.
    You can add a case by selecting the "Case Management" tab on the top left of the page. This page will allow you to add, edit, and remove cases.
    Clicking on the "+Add Case" button on the top left will open a new panel which will allow you to select which workflows and which execution elements you wish to log. There is only one controller, and you can see the playbooks which are loaded into the controller.
    If you double click on an element, you'll see it expand into its component elements. If you click on each element, you'll see a list of events on the right side of the panel. These boxes indicate which events will be logged to this case for this element.
