Workflow Editor Tutorial

  • Navigate to Playbook Page

    Click the Playbook link on the left


  • Create New Workflow

    Click the New Workflow button


  • Enter New Workflow Name

    1. Enter MyPlaybook in the Playbook field.
    2. Enter MyWorkflow in the Workflow field
    3. Click Create Workflow

  • Open Workflow in Editor

    Double click the MyWorkflow workflow now shown on the left


  • Show Action Tab

    Click the Action tab to show the list of available actions


  • Drag and Drop helloWorld action into Graph


  • Drag and Drop pause Action into Graph


  • Drag and Drop repeatBackToMe Action into Graph


  • Drag and Drop returnPlusOne Action into Graph


  • Drag and Drop getQuote Action into Graph


  • Create Edge 1

    Link two nodes as shown by dragging from the red circle at the top of the node

    Link two nodes as shown by dragging from the red circle at the top of the node

    Link two nodes as shown by dragging from the red circle at the top of the node

  • Create Edge 5

    Link two nodes as shown by dragging from the red circle at the top of the node


  • Save Workflow

    Press the Save Workflow button to save the workflow on the server.


  • Select Node

    Click the node to select. It will turn blue and node properties will be shown to the right of the graph


  • Delete Node

    Press the Delete button to delete the selected node. Can be used to delete edges as well.


  • Undo

    Press the Undo button to undo the deletion of the previous step. The node will now appear.


  • Drag Node

    Nodes can dragged as shown by clicking on a node with the left mouse button and and moving the mouse with the left mouse button still pressed.


  • Edit Node Parameters

    You may edit the node parameters to the right of the graph as desired.


  • Pan

    You may shift the entire graph in any direction by either (1) dragging with mouse by clicking anywhere in graph where there is no node or edge, or (2) using the pan control on the top right of the graph.


  • Zoom

    You may zoom the entire graph in or out by either (1) using the mousewheel, or (2) use the zoom control on the top right of the graph.


  • Reset

    Press the Reset button to revert the graph to the default zoom. This undos any zooming or panning done previously.


  • Rename, Duplicate, or Delete Playbook or Workflow

    To rename, duplicate, or delete the playbook or workflow, right-click the playbook or workflow and press the "Rename Playbook", "Rename Workflow", "Duplicate Playbook", "Duplicate Workflow", "Delete Playbook", or "Delete Workflow" options.
