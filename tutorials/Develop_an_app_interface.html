Develop an app interface

1. Under the Python package with the name of your app, create three directories called interface, static, and templates.

In this example we will be recreating the some of the functions of the HelloWorld App in the apps directory, so we'll call our package HelloWorld. The apps directory should look like this:


WALKOFF
|-- apps
  |-- __init__.py
  |-- HelloWorld
    |-- interface
      |-- static
      |-- templates
    |-- __init__.py
  |-- [Other apps]

2. Create a basic Display module

Inside of your app Python package, we need to create a display.py module. Inside of this module, we will define a method called load which can allow you to pass an abritrary number of arguements. A sample display.py module would look like this:


def load(*args, **kwargs):
    return {}

3. Create a simple web page

Inside of the templates directory, we create a index.html page.

4. Create a basic main application script

Inside of the static directory, we create a main.js script. Inside of this script, we can write code that uses the jQuery and other helper plugins.

Congratulations! You now have made an interface for your first WALKOFF app.

Go have a look in the webapp and see the name of your app included in the "Installed Apps" portion of the dashboard and see the interface you created for your app. Next, you'll learn how to test your app using a top-level test script.
