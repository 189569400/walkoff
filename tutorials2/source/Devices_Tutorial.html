Devices Tutorial

  • Add device

    1. Select the app you want to modify
    2. Fill out the parameters form
    3. Click the add device button

  • Add device

    1. Select the app you want to modify
    2. Select the device you want to modify
    3. Fill out the parameter form
    4. Click Edit

  • Remove device

    1. Select the app you that contains the device to remove
    2. Select the device you want to remove
    3. Click the Remove button
